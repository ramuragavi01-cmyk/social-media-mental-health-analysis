# Social Media & Mental Health Analysis

## Project Overview
This project analyzes the relationship between social media usage and mental health using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Social Media and Mental Health Survey (481 responses)

## Analysis
- Gender Distribution
- Age Distribution
- Most Used Platforms
- Time Spent on Social Media
- Depression Analysis
- Concentration Analysis
- Sleep Issues Analysis

## Key Findings
- Most respondents are aged 18–25.
- Instagram and YouTube are the most popular platforms.
- Higher daily social media usage is associated with increased depression and sleep issues.
